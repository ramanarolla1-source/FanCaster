# FanCaster Architecture

## Overview

FanCaster transforms trusted sports data into publication-ready content using TxLINE and QVAC Local AI.

```
Live Sports Events
        │
        ▼
 TxLINE Sports Data
        │
        ▼
 Editorial Intelligence
        │
        ▼
   QVAC Local AI
        │
        ▼
 Published Content
```

## Components

### TxLINE

Provides trusted sports data including:

- Live events
- Scores
- Fixtures
- Historical data
- Consensus betting odds

### Editorial Intelligence

Processes incoming sports events to identify content opportunities and determine their editorial significance.

### QVAC Local AI

Generates publication-ready content such as:

- Social media posts
- Match summaries
- Headlines
- Reports
- Video scripts

## Output

Generated content can be published across:

- X
- Instagram
- Blogs
- Sports websites

---

FanCaster is modular, allowing additional sports, data providers, and publishing platforms to be integrated with minimal changes.
